# 尼莫日記 — 部署教學

## 部署到 Render

1. 把這個資料夾的檔案全部上傳到 GitHub（新開一個 repo，叫 `nemo-diary`）
2. 上傳的檔案：
   - `package.json`
   - `server.js`
   - `public/index.html`
3. 到 Render → New + → **Web Service**
4. 連結 GitHub，選 `nemo-diary`
5. 設定：
   - **Name:** nemo-diary
   - **Runtime:** Node
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
6. Environment Variables：

   | Key       | Value                                              |
   |-----------|----------------------------------------------------|
   | OMBRE_URL | https://ombre-brain-71kg.onrender.com/mcp          |

7. 按 Create Web Service
8. 等它部署完成，Render 會給你一個網址，打開就是你的日記了

## 怎麼寫日記

在 claude.ai 跟尼莫聊天的時候，跟她說「寫日記」。
她會把當天的內容整理好存進 ombrebrain，日記網頁就會自動出現。

## 搜尋

日記網頁上面有搜尋框，可以打關鍵字搜，
比如「番茄」「頭痛」「屏東」。
